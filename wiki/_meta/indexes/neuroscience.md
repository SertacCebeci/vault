## Concepts (30)

- [[active-learning-declarative]] — Active learning from a neuroscience perspective: cognitive engagement that builds and consolidates neural links via the declarative pathway
- [[bdnf-and-exercise]] — Exercise produces BDNF for dendritic spine growth; neurogenesis; stress buffering for cognition
- [[biologically-primary-and-secondary-knowledge]] — Evolutionary framework: primary abilities (faces, language) are pre-wired; secondary abilities (reading, math) require deliberate neural repurposing
- [[brain-breaks]] — Strategic pauses during instruction enabling hippocampal offloading and micro-consolidation
- [[brain-development-timeline]] — Neuropil formation, synaptic pruning, back-to-front maturation from infancy through adolescence
- [[chunking]] — Binding information into compact neural patterns through focused attention, understanding, and context
- [[consolidation]] — Brain reorganizes and stabilizes memory traces into long-term memory; essay-writing analogy
- [[declarative-learning-system]] — The three-part neural architecture for conscious learning: working memory (conductor) → hippocampus (indexer) → neocortex (store)
- [[declarative-procedural-seesaw]] — The two learning systems act like a seesaw; learning through both produces the most powerful knowledge
- [[dopamine-and-learning]] — Dopamine from unexpected rewards strengthens synaptic connections and temporarily enables declarative-procedural cooperation
- [[encoding]] — First step of learning: converting perceptions into memory traces in short-term memory
- [[eustress-and-learning]] — Moderate transient stress enhances cognition and learning via glucocorticoids; inverted-U curve
- [[focused-and-diffuse-modes]] — The brain's two complementary thinking modes: tight/concentrated vs. relaxed/big-picture
- [[hebbian-learning]] — "Neurons that fire together wire together" — the synaptic mechanism underlying all learning
- [[hippocampus-as-index]] — The hippocampus functions as an index to the neocortex, not as storage; limited capacity, dual-direction constraint
- [[learn-it-link-it]] — Two-phase learning framework: initial neural connections (learn it) → strengthening and extending through practice (link it)
- [[long-term-memory]] — Vast distributed storage requiring consolidation via hippocampus; reconsolidation on recall
- [[long-term-memory-boosts-working-memory]] — LTM links extend apparent WM capacity through chunking without changing the underlying neural limit
- [[neuromodulators]] — Acetylcholine (attention), dopamine (motivation/reward), serotonin (social behavior)
- [[neuronal-recycling-hypothesis]] — Dehaene's theory: secondary skills repurpose brain regions evolved for related primary functions; constrained plasticity
- [[neuroplasticity]] — The brain remains modifiable through experience, practice, injury, and use
- [[procedural-front-and-back-door]] — Two entry points to the procedural system: goal-directed (front) and habitual (back)
- [[procedural-learning-system]] — The brain's second learning pathway via basal ganglia; nonconscious, slow to learn, fast to deploy
- [[procrastination]] — Pain-triggered avoidance operating through the habit loop (cue → routine → reward → belief)
- [[reconsolidation]] — Retrieval makes learning pliable again for updating and strengthening
- [[sleep-and-learning]] — Toxin clearance, memory consolidation, and neural pattern rehearsal during sleep
- [[social-buffering]] — Supportive group members moderate stress hormones to optimal learning zone
- [[temporal-discounting]] — Brains discount delayed rewards; steeper discounting in teenagers linked to the striatum
- [[top-down-and-bottom-up-attention]] — Two attentional pathways: voluntary prefrontal (top-down) vs. involuntary sensory-driven (bottom-up)
- [[working-memory]] — The ~4-slot prefrontal cortex system for immediate conscious processing
