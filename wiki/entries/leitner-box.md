# Leitner Box

The Leitner box is a simple flashcard-based system for implementing [[spaced-repetition]] without any technology. Developed by German scientist Sebastian Leitner, it uses a series of four (or more) file-card boxes to regulate how often each item is practiced based on current mastery.

## How it works

All new study materials — flashcards for vocabulary, musical scores, hockey moves, formulas — start in the first box and are practiced frequently because you make the most mistakes on them. When you answer a card correctly, it moves to the second box, which is practiced less often (perhaps half as frequently as the first). Correct answers in the second box promote the card to the third box, practiced even less often, and so on. If you miss a card at any level — get the question wrong, flub the music, miss the one-touch pass — it moves back up a box so you practice it more often.

The underlying principle is that *the better your mastery, the less frequent the practice* — but the item should *never disappear completely* from your practice rotation. This operationalizes the [[forgetting-curve]]: items you know well need only occasional retrieval to stay consolidated, while items you struggle with need frequent reinforcement.

## Concepts applied

- [[spaced-repetition]] — the box system automates increasing intervals between practice of mastered items
- [[desirable-difficulty]] — retrieving from memory at longer intervals is harder, which strengthens the memory trace
- [[recall-as-learning]] — each practice session requires active retrieval, not passive review
