# Hard-Start Technique

The hard-start technique is a test-taking strategy in which students begin by scanning for the hardest question, attempt it for a minute or two, then deliberately pull away to work on easier problems. The brain's [[focused-and-diffuse-modes|diffuse mode]] continues processing the hard problem in the background while the student makes progress elsewhere.

## How it works

When a student first engages a hard problem, focused mode loads the problem's structure into working memory. Switching to an easier question releases focused mode's grip, allowing diffuse mode to work subconsciously on the hard problem — testing distant neural connections, exploring alternative solution paths. When the student returns to the hard problem later, they often find unexpected progress or even a clear solution.

This is far more effective than the conventional strategy of starting with the easiest questions. By the time a conventionally-paced student reaches the hardest question at the end, they are mentally tired, stressed by dwindling time, and have no diffuse-mode processing to draw on. The hard-start approach uses the brain as a dual processor — focused mode on the current question, diffuse mode on the hard one.

The prerequisite is adequate preparation: the technique leverages existing knowledge by giving diffuse mode raw material to work with. An unprepared student gains nothing from attempting a hard problem first — they should simply start with easy questions and collect what points they can.
