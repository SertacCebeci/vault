# Expertise Reversal Effect

As learners gain knowledge in a domain, the instructional scaffolding that once helped them can start to *impede* their learning. Guidance that reduces [[working-memory]] load for novices becomes redundant processing for experts, consuming cognitive resources without adding value. The more knowledgeable students are, the less guidance they need — and too much guidance actively hurts.

## Elaboration

The effect arises from the interaction between [[working-memory]] and [[long-term-memory]]. Novices lack long-term memory structures for a topic, so their working memory bears the full processing burden — detailed guidance, [[worked-example-effect|worked examples]], and step-by-step scaffolding help by reducing that load. But as learners build [[long-term-memory]] links, their prior knowledge begins doing the work that scaffolding once did. At this point, the scaffolding itself becomes extraneous information that working memory must process alongside the actual content, creating unnecessary [[chunking|cognitive load]].

This has direct implications for [[scaffolded-instruction]]: effective differentiation requires not just adding support for struggling students but also *removing* support for advanced ones. [[race-car-and-hiker-brains|Race-car learners]] may need complex, multilayered problems rather than more of the same guided exercises. The goal is to match instructional support to the learner's current knowledge state, which shifts as learning progresses.

The effect also explains why education's transformative power lies not in changing students' working memory capacity (which is largely fixed) but in changing the amount of knowledge held in long-term memory, which in turn extends apparent working memory capacity through [[long-term-memory-boosts-working-memory]].
