# Lens — Application

## Question

Could a practitioner follow this as instructions? Does it have steps, timing, or implementation details?

## How to apply

An application is a concrete intervention — what a teacher, learner, or system builder actually does in the world. It stands on a foundation of concepts (the `applies` frontmatter field lists them) but its own body is prescriptive, not explanatory.

Distinguishing from concept: the concept note for `recall-as-learning` explains *why* retrieval strengthens memory; the application note for `low-stakes-classroom-quizzing` explains *how* to run such quizzes — frequency, format, grading, feedback.

Applications can cite evidence for why they work, but evidence supports the technique, not the categorization.
